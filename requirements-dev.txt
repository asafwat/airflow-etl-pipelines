# Dev dependencies for local DAG development + CI.
#
# IMPORTANT: when installing these, also use Airflow's official constraints
# file to pin transitive deps to the versions Airflow tested against —
# without it, pip resolves to latest versions which breaks Airflow internals
# (e.g. structlog 25 removed `Styles` that Airflow 3.1.0's logging imports):
#
#   AIRFLOW_VERSION=3.1.0
#   PYTHON_VERSION=3.12
#   CONSTRAINTS="https://raw.githubusercontent.com/apache/airflow/constraints-${AIRFLOW_VERSION}/constraints-${PYTHON_VERSION}.txt"
#   pip install -r requirements-dev.txt --constraint "${CONSTRAINTS}"
#
# CI does this for you (see .github/workflows/ci.yml).
